### Run local development

1. Clone repository
    ```
    git clone https://github.com/dstopia/dstopia.git
    
    cd dstopia
    ```
2. Install all dependencies
    ```
    npm install
    ```
3. Copy file `.env.example` and rename into `.env`
4. Run server in development
    ```
    npm run dev
    ```
